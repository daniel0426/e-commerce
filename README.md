## 1. Purpose of this projecct

- To make real e-commerce application using commerce.js and stripe
- Practice Material UI for styling

## 2. Tech Stack 

1. React.js 
2. Commerce.js
3. Stripe
4. Material UI 



## 3. 💥 Issue
